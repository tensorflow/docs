tensorflow::ops::L2Loss

#include <nn_ops.h>

L2 Loss.

Summary

Computes half the L2 norm of a tensor without the sqrt: 

output = sum(t ** 2) / 2

Arguments:

  • scope: A Scope object
  • t: Typically 2-D, but may have any dimensions.

Returns:

  • Output: 0-D.
