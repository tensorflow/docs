tensorflow::ops::Sigmoid

#include <math_ops.h>

Computes sigmoid of x element-wise.

Summary

Specifically, y = 1 / (1 + exp(-x)).

Arguments:

  • scope: A Scope object

Returns:

  • Output: The y tensor.
